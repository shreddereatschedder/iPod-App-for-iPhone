# iPod-App-for-iPhone

Repository scaffold for the iPod-inspired iPhone app.

Structure:

iPod-App-for-iPhone/
├── .github/
│   └── workflows/
│       └── build.yml
├── iPodApp/
│   ├── App/
│   │   └── iPodApp.swift
│   ├── Views/
│   │   ├── ContentView.swift
│   │   ├── LibraryView.swift
│   │   ├── PlayerView.swift
│   │   ├── PlaylistView.swift
│   │   └── SearchView.swift
│   ├── Models/
│   │   ├── Song.swift
│   │   └── Playlist.swift
│   ├── Services/
│   │   ├── YouTubeService.swift
│   │   ├── DownloadService.swift
│   │   └── AudioPlayerService.swift
│   └── Info.plist
├── project.yml
└── README.md

You can open the `iPodApp` folder in Xcode or integrate it into your existing project.
